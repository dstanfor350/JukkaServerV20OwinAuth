JukkaOWINTest is intended to be a test driver of JukkaServer20OWINAuth.
Make JukkaServer20OWINAuth the startup program and click the IIS Express button in the debug toolbar to start the server.  
A browser window should appear with a "HTTP Error 403.14 - Forbidden", this is OK
The address bar will have localhost:<port number>

The start the JukkaOWINTest by right-clicking the project, select 'debug', then 'start new instance'.
As long as the browser window shows the JukkaOWINTest can be return.
